Veritate Fact Check API

agregador e API para checagem de fatos no Brasil

fact check aggregator for Brazilian news and politics

Este endereço é apenas o acesso à API do Veritate Fact Check. A API é usado no site Veritate ( veritate.wowperations.com.br), um projeto experimental e educacional sem fins lucrativos de agregador/buscador reunindo artigos diversas agências de checagem de fatos como Agência Pública, Aos Fatos e Lupa, entre outros.

Os objetivos do projeto são:

  1. Investigar, estudar e experimentar tecnologias, algoritmo, padrões de design e experiência de uso em buscas e jornalismo intermediado por algoritmo
  2. Fomentar o consumo de jornalismo de qualidade e o compartilhamento de checagem de fatos e instigar o senso crítico contra notícias falsas ( Fake News)

Para mais informações, para solicitar a inclusão de um site de checagem ou solicitar exclusão de material que viole direitos autorais, entre em contato através do email veritate@wowperations.com.br
